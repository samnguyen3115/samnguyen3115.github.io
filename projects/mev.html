MEV — Math Expression Validator

Lightweight, Unit-Aware Symbolic Expression Framework in Python

About the Project

MEV is a lightweight, unit-aware symbolic expression framework in Python. It constructs expression Abstract Syntax Trees (ASTs), executes safety validation checks against runtime data contexts, and evaluates ONLY when validation succeeds.

By ensuring a validation-first pipeline, MEV prevents partial side-effects from runtime errors like division-by-zero, invalid mathematical domains (e.g., negative square roots), or incompatible unit conversions.

Key Features

  • Validation-First Pipeline - Traverses the AST to verify all variables are present, division-by-zero is avoided, and domain limits are satisfied before running evaluations.
  • Physical Unit Awareness - Integrates directly with Pint to scale, normalize, and validate compatible physical units (e.g. cm/s to mph).
  • Extensible Registry - Registry mapping mathematical functions to custom implementations and domain rules without altering the core computation engine.
  • AST Visualization - Supports exporting expression hierarchies as text-based tree structures or Mermaid flowcharts for debugging.
