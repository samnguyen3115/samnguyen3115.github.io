EventEase

AI-Powered Intelligent Event & Task Planner

About the Project

EventEase is an intelligent, AI-powered event management and task orchestration platform designed to simplify complex planning processes. Built with Flask and powered by Google's Gemini AI, it helps users automatically break down descriptions of events into actionable, contextual checklists.

Planning large events—like conferences, parties, or meetups—involves dozens of hidden details. EventEase removes the guesswork by using generative AI to construct smart, customized checklists in real time. It also features computer vision integrations to verify task completion and collaborative workflows for multi-user coordination.

Key Features

  • AI-Powered Task Generation - Natural language conversational interface (via Google Gemini) that parses event parameters to generate tailored lists of tasks.
  • Computer Vision Verification - Integrates with Salesforce BLIP model to analyze user-uploaded photos for automated task verification.
  • Real-time Collaboration - Multi-user planning workspace with role-based access and automated mail notifications for team invitations.
  • Dynamic Progress Dashboard - Visual progress bars, analytics, and task tracking interfaces built using Bootstrap 5.
  • Calendar Integration - Seamless export to Apple, Google, and Outlook calendars via standardized ICS files.
  • Multi-language Support - Built-in translation module to translate planning tasks dynamically for international teams.
